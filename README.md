# 🔐 Smart IoT Security Monitor using AI

A simple cybersecurity project that combines **IoT simulation, machine learning, and anomaly detection** to monitor device activity and detect suspicious behavior.

This project demonstrates how **AI can be used to enhance IoT security** by identifying abnormal traffic patterns from connected devices.

---

## 📌 Project Overview

IoT devices such as smart cameras, sensors, and smart lights are widely used in homes and industries. However, many IoT devices lack strong security mechanisms and can be vulnerable to cyber attacks.

This project simulates IoT device traffic and uses a **machine learning model to detect unusual activity** that may indicate potential cyber threats.

The system monitors device behavior and generates alerts when suspicious patterns are detected.

---

## 🎯 Objectives

* Simulate activity of IoT devices.
* Monitor network traffic generated by these devices.
* Train a machine learning model to learn normal device behavior.
* Detect anomalies or suspicious activity in device traffic.
* Generate alerts when abnormal behavior is identified.

## 🤖 Machine Learning Model

The project uses the **Isolation Forest algorithm** to detect anomalies in IoT device traffic.

### Steps

1. Generate simulated normal device traffic
2. Train the ML model on normal behavior
3. Introduce abnormal traffic patterns
4. Detect suspicious activity using anomaly detection

---

## ▶️ Running the Project

### Step 1 – Run IoT Device Simulator

```
python simulation/device_simulator.py
```

This will generate simulated IoT device traffic data.

---

### Step 2 – Train the Machine Learning Model

```
python ml_model/anomaly_detector.py
```

This trains the anomaly detection model using the simulated data.

---

### Step 3 – Start the Backend Server

```
python backend/app.py
```

The Flask server will start and monitor incoming device activity.

---

### Step 4 – Run Visualization Dashboard

```
python visualization/dashboard.py
```

This will display graphs showing IoT device traffic and anomaly detection.
